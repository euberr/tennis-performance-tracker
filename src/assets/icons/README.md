# PWA Icons

Questa cartella deve contenere le icone per la Progressive Web App.

## Dimensioni richieste

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png (richiesto)
- icon-384x384.png
- icon-512x512.png (richiesto)

## Come generare le icone

1. Prepara un'icona base quadrata (almeno 512x512px)
2. Usa uno di questi strumenti:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
   - https://www.favicon-generator.org/
3. Scarica tutte le dimensioni e mettile in questa cartella

## Placeholder temporaneo

Se non hai ancora le icone, puoi creare placeholder semplici usando:
- Un tool online come https://dummyimage.com/
- Un editor grafico (GIMP, Photoshop, etc.)
- Un generatore SVG convertito in PNG

L'icona dovrebbe rappresentare un tema tennis (racchetta, pallina, campo, etc.) con colori che corrispondono al tema dell'app (#0B3D2E, #F6E547).
